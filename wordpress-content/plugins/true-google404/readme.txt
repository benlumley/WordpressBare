=== True Google 404===Contributors: technofreakPlugin Name: True Google 404Plugin URI: http://thetechnofreak.com/technofreak/wordpress-plugin-true-google-404/Tags: 404, search, google, error, page not found, missing, seo, 404 log, reportDonate link: http://thetechnofreak.comRequires at least: 2.7Tested up to: 3.5.1Stable tag: 1.4.1Transform a 404 to a list of related links with descriptions & view 404 logs.Don't irritate readers with dumb 404s, keep'em with you== Description ==Using the true [Google Search Parser](http://thetechnofreak.com/true-google-search-scraper-parser/ "True Google Search API") API by [The Technofreak](http://TheTechnofreak.com "Technology, Web Development, Graphics Design, Gadgets at Their Best") , this plugin will intercept your standard 404 page and return a list of urls that may help your user find the content they are looking for.This plugin also logs any and all 404 hits so that you can improve on them, with comprehensive information about the requested URL, the number of suggestions shown for that page, the referrer (i.e. the source) from where the user came, the time of hit and the keywords extracted for that page.You can limit the search to just the content on your site, set a default prefix to the search query (for example in case you want to limit the search to some keywords). This provides your users with quality recomendations.And keep your readers with your site and does not irritate them with dumb 404s.This plugin should work out-of-the-box with most themes using the included 404 template.You can also define your own to fine tune the look and feel to match your theme.== Installation ==* Download the plugin with the link at right and upload the plugin to your blog **OR*** simply go to **plugins->add new->search** on *your* wordpress website and search for "*true google 404*". Click on "*install now*"1. Activate the plugin.1. Go to true Google 404 settings and tweak the settings as you like.1. If you want to use the plugin while using your theme's **404.php** (You can find it at: Appearance->Edit theme->and click on 404.php in the right bar), add the line:`<?php if(function_exists('trueGoogle404'))echo trueGoogle404();?>` Wherever you want the results to show.1. You're golden. Congrats! No more flop 404's1. See the 404 logs at **Tools->404 Stats**1. Review and configure settings at **Settings->true Google 404**== Frequently Asked Questions === Don't you have a live example of this plugin? =No, we have it. Try it out by typing a non existant url on my website.For example go to http://thetechnofreak.com/my-wordpress-pluginor try any other path instead of [my-wordpress-plugin] to see the plugin in action.= I am no more getting any suggestions on a 404 page. Why? =Your server's IP address probably got blocked by google for excessive queries. Contact the plugin author to get additional functionality of using proxies. = No suggestions are shown on my 404 pages =Your website possible is not indexed fully by google= No referrals are shown in the log for 404 pages =The visitor came directly to the page (by typing the url in address bar), or the visitors browser is set to not to disclose this information.== Screenshots ==1.
